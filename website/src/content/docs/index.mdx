## Install

```bash
npm install -g @wavyx/hscli
```

Requires Node.js 20+. The command is `hscli`.

## Built for agents and automation

Because every command speaks JSON and returns clean [exit codes](/hscli/commands/),
an AI coding agent or a shell script can operate Help Scout end to end:

```bash
# Triage the active inbox, then close everything already tagged "resolved"
hscli conv list --status active --output json \
  --jq '.[] | select(.tags[]? == "resolved") | .id' \
  | xargs -I {} hscli conv status {} --set closed
```

Point an agent at `hscli --help` (or this reference) and it has a complete,
self-describing toolset for conversations, customers, reporting, and backups.
