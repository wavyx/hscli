Every command is built from the same grammar:

```text
hscli <group> <action> [target] [flags]
```

Run `hscli <group> --help` for the live, self-describing version of any group.

## conv <span class="badge badge--accent badge--dot">conversations</span>

List, read, and act on conversations — the workhorse of hscli.
See the [Conversations guide](/guides/conversations/) for worked examples.

| Command | Description | Key flags |
| --- | --- | --- |
| `conv list` | List conversations in an inbox. | `--status` `--tag` `--assigned` |
| `conv get <id>` | Fetch a single conversation with threads. | `--threads` |
| `conv reply <id>` | Post a reply or note. | `--note` `--body` |
| `conv status <id>` | Change status (`active` · `pending` · `closed`). | `--set` |
| `conv tag <id>` | Add or remove tags. | `--add` `--remove` |
| `conv snooze <id>` | Snooze until a time. | `--until` `--stdin` |

## api <span class="badge badge--coral badge--dot">escape hatch</span>

Reach any Help Scout endpoint directly — locked to the Help Scout host. See the
[`hscli api` reference](/reference/api/).

```bash frame="terminal"
hscli api GET /v2/mailboxes --jq '.[] | .name'
hscli api POST /v2/conversations/4831/tags --data '{"tags":["vip"]}'
```

## Global flags

These work on every command.

| Flag | Description | Default |
| --- | --- | --- |
| `--output` | Output format (`table` · `json` · `yaml` · `csv`). | `table` |
| `--jq` | Apply a jq expression to JSON output. | — |
| `--fields` | Project a comma-separated subset of fields. | all |
| `--profile` | Use a named auth profile. | `default` |
| `--quiet` | Suppress non-data output. | `false` |
| `--help` | Self-describing help for any command. | — |

See [Output & filtering](/automation/output/) for details.

## Exit codes

Deterministic, so scripts and agents can branch on the result.

| Code | Meaning |
| --- | --- |
| `0` | <span class="badge badge--lime badge--dot">success</span> |
| `1` | Usage / argument error |
| `2` | Authentication error |
| `4` | Resource not found |
| `5` | Rate limited — retry with backoff |
| `6` | Help Scout API error (5xx) |

Full guide: [Exit codes](/automation/exit-codes/).
