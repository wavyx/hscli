---
title: Customers
description: Search, read, create, and update Help Scout customer records.
---

The `customer` group works with the people behind your conversations. See the
[command reference](/reference/commands/#customer) for all flags.

## Search and list

```bash frame="terminal" title="zsh — hscli"
hscli customer search "acme"
hscli customer list --mailbox 42 --output json
```

`customer list` filters by `--mailbox`, `--first`, `--last`, or `--since`; use
`customer search "<query>"` for free-text lookup.

## Read a profile

```bash frame="terminal"
hscli customer get 99120 --output yaml
```

## Create and update

```bash frame="terminal"
# create
hscli customer create --email kai@studio.co --first Kai --last Okafor --company "Studio Co"

# update an existing record
hscli customer update 99120 --company "Studio Collective" --job-title "Founder"
```

## A customer's conversations

```bash frame="terminal"
hscli customer conversations 99120 --limit 10
```

:::tip
Pipe `customer search` into `--jq` to pull just the IDs, then feed them to
`customer conversations` or `conv` commands.
:::
