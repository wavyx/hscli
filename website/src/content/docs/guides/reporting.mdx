---
title: Reporting
description: Pull conversation, company, and per-user metrics as JSON, CSV, or a table.
---

The `report` group turns Help Scout's reporting into data you can pipe anywhere. It's a
set of focused reports, each scoped by a date range with `--start` / `--end`. See the
[command reference](/reference/commands/#report) for all flags.

## Conversation volume & resolution

```bash frame="terminal" title="zsh — hscli"
$ hscli report conversations --start 2026-05-01 --end 2026-05-31 --output json
[
  { "mailbox": "Support", "conversations": 1284, "resolved": 1102 },
  { "mailbox": "Billing", "conversations": 331,  "resolved": 318  }
]
```

## Company-wide rollup

```bash frame="terminal"
hscli report company --start 2026-05-01 --end 2026-05-31
```

## Per-agent productivity

```bash frame="terminal"
hscli report user --start 2026-05-01 --end 2026-05-31 --user 10
```

## Pull metrics into your stack

Reports return nested JSON — stream them into a notebook, a BI tool, or a `jq` pipeline:

```bash frame="terminal"
hscli report conversations --start 2026-05-01 --end 2026-05-31 --output json > may.json
```

:::note
Reports are nested, so `--output csv`/`table` aren't supported here — use `json` or
`yaml`, and reshape with `--jq` when you need a flat CSV.
:::

:::tip
Schedule this in [CI](/automation/ci/) to keep a dashboard fresh.
:::
